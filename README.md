# 📿 My Dua Library

A clean, elegant, and offline-capable web application designed to help you manage and recite your personal collection of Duas (supplications). 

## ✨ Features

- **Personalized Library**: Add, edit, and delete your own Duas.
- **Offline Persistence**: Uses **IndexedDB** to store your data locally in the browser—no internet connection or server required.
- **Smart Search**: Quickly find Duas by searching through titles or translations.
- **Beautiful Typography**: Optimized for readability with specialized fonts:
    - *Inter* for English text.
    - *Noto Naskh Arabic* for clear and traditional Arabic script.
- **One-Click Copy**: Easily copy Arabic text or translations to your clipboard for sharing.
- **Responsive Design**: Fully functional on mobile, tablet, and desktop devices.
- **Categorization**: Organize your Duas into categories like Daily, Morning, Evening, Protection, etc.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Modern styling with Flexbox, Grid, and CSS Transitions.
- **JavaScript (ES6+)**: Vanilla JS for logic and DOM manipulation.
- **IndexedDB**: Low-level API for client-side storage of significant amounts of structured data.
- **Google Fonts**: Integration for high-quality typography.

## 🚀 Getting Started

Since this is a client-side application, there is no installation required.

1.  Save the code as `index.html`.
2.  Open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari).
3.  Start adding your Duas!

## 📖 How to Use

1.  **View a Dua**: Click on any card in the main list to open the detail view.
2.  **Add New**: Click the floating `+` button at the bottom right to open the "Add New Dua" form.
3.  **Search**: Use the search bar at the top to filter your collection in real-time.
4.  **Edit/Delete**: Inside the detail view, use the ✏️ (pencil) icon to edit or the 🗑️ (trash) icon to remove a Dua.
5.  **Copy**: Use the "Copy Arabic" or "Copy Translation" buttons to save the text to your clipboard.

## 🔒 Privacy & Data

All data is stored locally on your device using IndexedDB. No data is sent to any server, ensuring your personal collection remains private. Note that clearing your browser's site data/cache may remove your stored Duas unless you have backed them up.
